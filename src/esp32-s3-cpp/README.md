# Guardian ESP32-S3 C++ Firmware

This project contains the implementation of the chainsaw detection firmware for the ESP32-S3 hardware platform.

## Features
- **3-Stage Power Cascade**: Optimized for sub-1mA average current draw.
- **I2S DMA Audio Capture**: High-fidelity audio sampling.
- **ESP-DSP Spectral Analysis**: Preliminary pattern verification.
- **Edge Impulse AI Integration**: Real-time sound classification with TFLite Micro.
- **LoRa Mesh Connectivity**: Alert propagation across the forest network.
